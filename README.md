# Molecular and Spatial Organization of the Primary Olfactory System

This repository contains MERFISHEYES code tools for the research on the molecular and spatial organization of the mouse primary olfactory system, as described in:

**Title**: Molecular and Spatial Organization of the Primary Olfactory System and its Responses to Social Odors  
**Authors**: Bogdan Bintu, Yoh Isogai, Ignatius Jenie, Xiaowei Zhuang, Catherine Dulac  
**Journal**: Cell (2026)  
**DOI**: [10.1016/j.cell.2026.03.053](https://www.cell.com/cell/fulltext/S0092-8674(26)00389-2)

## Project Overview

This research investigates how the mouse olfactory system processes sensory information, focusing on:

- Comprehensive mapping of odorant receptor (OR) expression in the main olfactory epithelium (MOE) and olfactory bulb (OB)
- Analysis of sensory neuron distribution patterns
- Identification of spatial domains for olfactory responses
- Molecular basis of sensory processing in the olfactory system
